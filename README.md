Tic Tac Toe - Python Pygame
This is a simple and interactive Tic Tac Toe game built with Python and Pygame.  
The game features a classic 3x3 grid where the player competes against the computer.

Game Features
- Classic Tic Tac Toe gameplay on a 3x3 board.  
- Player vs Computer mode with simple AI.  
- Turn-based game with visual highlights and animations.  
- Reset the game anytime by pressing the **R** key after a game ends.

Controls
- Click on any empty cell to place your mark (X).  
- Computer will automatically play after your turn (O).  
- Press **R** to reset the game after it finishes.
